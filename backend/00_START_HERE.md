## 💡 Cas d'Usage Typiques

### Matin - Check Daily (5 min)
```
1. Voir KPI Dashboard → État du support
2. Si Critiques → Chercher et traiter
3. Sinon → Filtrer "À traiter" + Tri "Urgent"
4. Commencer journée
```

### Recherche Rapide (1 min)
```
1. Barre recherche
2. Taper nom centre ou admin
3. Résultat immédiat
4. Traiter le ticket
```

### Fin de Journée (5 min)
```
1. Filtrer "En traitement"
2. Vérifier terminé/non-terminé
3. Marquer traité ceux finis
4. Sauvegarder session
```
