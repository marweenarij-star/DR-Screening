## 🎥 Quick Reference Cards

### KPI Dashboard (30 sec)
```
📊 4 indicateurs clés:
  1. Total = nombre tickets
  2. Temps moyen = vitesse traitement
  3. Critiques = haute priorité urgent
  4. Taux résolution = % traité vs total
```

### Recherche (30 sec)
```
🔍 Tape pour chercher:
  ✓ Sujet du ticket
  ✓ Contenu message
  ✓ Nom centre
  ✓ Nom administrateur
→ Résultats instantanés
```

### Filtres (30 sec)
```
🎛️ 4 filtres disponibles:
  ✓ Statut (À traiter/En cours/Traité)
  ✓ Priorité (Haute/Normale/Basse)
  ✓ Tri (Récent/Urgent/Ancien)
  ✓ Bouton Réinitialiser
```

### Actions (30 sec)
```
✏️ 3 actions par ticket:
  ✓ À traiter (green)
  ✓ En traitement (orange)
  ✓ Traité (gray)
→ Cliquer = changer status
```
